# Installing Express.js

Simply open the Command Line Interface (CLI) / shell of your choice on your operating system. Below are a few common ones:

* Linux: Bash
* MacOS: Terminal
* Windows: Terminal, Command Prompt, Windows Powershell

Then navigate to this directory in one of the two following ways. Option one is for Windows only while option two will work on Linux, MacOS, and Windows:

1) Run a terminal command from the address bar of file explorer by typing in the command to the address bar and hitting the ENTER key. Here are the commands listed for each shell option:

* For Command Prompt: cmd
* For Windows Powershell: powershell

2) Open Command Prompt (Windows), Powershell (Windows), Bash (Linux), or Terminal (MacOS), and manually navigate to this directory by typing the following command mentioned in the directory-mapping.txt file:

    `cd ../somefolder1/somefolder2/Username`

where "cd" is the command for "change directory" and "../somefolder1/somefolder2/Username" is the placeholder representation of the path to your Username folder. Please replace it with the actual path to your username folder. This can easily be found by double clicking the address bar of file explorer while being inside of the "Username" directory.

Now once you are inside of the directory in the terminal, run the following command:

`npm install express`

This will install express.js and allow you to run the "node server.js" command which will open your site locally on your computer on a localhost port that can be accessed through a browser of your choice. Do note that none of this will work or make any sense if you have not first installed node.js! If you have not done that then please visit this link: https://nodejs.org/en/download
