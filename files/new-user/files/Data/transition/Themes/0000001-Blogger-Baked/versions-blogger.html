Framework Versions
This is a complete directory of the Asveora framework's various versions. While some quality of life features may be added into older versions of Asveora to match operational standards, they will largely remain the same as when they were originally published. This is to ensure that Asveorans and anyone else using the Asveora framework have the freedom to choose which version of Asveora they prefer to deploy. Please keep in mind that some versions of Asveora might have shared features depending on the introduction of certain components and only the distinct elements of each version are separated for deployment. For example, v3 and v4 have completely different profile space layouts but otherwise share all of the same features with v4 having more features than v3. To find the latest releases, please visit Asveora's GitHub page where all repositories related to Asveora are maintained and archived.
