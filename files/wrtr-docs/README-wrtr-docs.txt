========================================
Understanding wrtr-docs
========================================

The HTML Document (Writer) format offered in LibreOffice (a fork of OpenOffice) allows users to write webpages in a traditional word-processing application environment. This means you can write a webpage in the same way that you would write a paper for school or work using an application like Microsoft Word. The best part? LibreOffice is open source and free to use. It has all of the same capabilities as the Microsoft Office suite with plenty of customizations to make your document writing experience easier. To download the LibreOffice suite, please head to their official website: https://www.libreoffice.org/

Once you have downloaded and configured LibreOffice on your system, you can get straight to work with the provided template files under the "PageName" directory. You will see two folders inside of the directory labeled "ODT-Drafting" and "HTML-Export" next to each other. Go on ahead and make a copy of the PageName directory and rename it after whatever page you are wanting to create. For example:

profile-home
%%%%ODT-Drafintg
%%%%HTML-Export

This would be a directory containing the documents you will edit and export for your "profile-home" HTML page. Now go into "ODT-Drafting" and edit the included ODT file as you would any normal word-processing document. Once you have written your page the way that you want it to be, simply go to:

File > "Save As..."

You will then be greeted with a file browsing window. Go on ahead and save your document as an ODF Text Document first with its filename. For this example we will call it "profile-home" and click the "Save" button. Now we will repeat the steps we followed to get to the "Save As..." file browsing window. Once back there again, we will click the "Save as tpye:" field and select "HTML Document (Writer)" from the list. This will enable us to save our document as an HTML file with any necessary media associations. Click the "Save" button and now you should see some new files generated (or just a single file if you only wrote a document with no media attachments in it such as images). Take those files (or file) and move them into the "HTML-Export" directory while keeping your original media files and ODF Text Document file in the "ODT-Drafting" directory. Now, simply zip your "HTML-Export" folder with a tool like 7-Zip (https://www.7-zip.org/) and send it to your wheelcore administrator for publishing to the web or publish it yourself!
